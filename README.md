# PHP-FPM 8.0

**Available Architectures:**  `amd64`, `arm64`

This repository will provide you a fully functional PHP-FPM 8.0 Docker image built from [official sources](https://github.com/php/php-src) nightly. It provides the base for [LinuxEye PHP-FPM Docker images](https://github.com/linuxeye/docker-php-fpm).

## 🐋 Available Docker tags

[![](https://img.shields.io/docker/pulls/bypanel/php-fpm-8.0.svg)](https://hub.docker.com/r/bypanel/php-fpm-8.0)

[`latest`][tag_latest] [`buster`][tag_buster]

```bash
docker pull bypanel/php-fpm-8.0
```
